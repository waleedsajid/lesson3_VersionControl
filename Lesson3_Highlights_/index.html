Highlight #1 $ git add.

This command updates the index using the current content found in the working tree, to prepare the content staged for the next commit. For example I used $ git add to add my files from git hub to cloud9.

Highlight #2 $ git status

The git status command displays the state of the working directory and the staging area. It lets you see which changes have been staged, which havenâ€™t, and the files that are not being tracked by Git.

Highlight #3 $ git init

This command creates an empty Git repository for objects, refs/heads, refs/tags, and template files. For example, I used $ git init to make sure files were empty.

Highlight #4 git hub

Git Hub is a website that is useful to for your online projects. Git hub allows you to make websites and also allows to transfer data to another program like cloud9.Git Hub is really good to host your very own website.

Highlight #5 $ git push

This command updates remote refs along with associated objects. For example if you have a repository and you use the push command, then the terminal will let you know what is going on.
